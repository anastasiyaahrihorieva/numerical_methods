## Содержание

1. Решение уравнений высших порядков
2. Решение алгебраических и трансцендентных уравнений
3. Решение СЛАУ
   - Прямые методы
   - Итерационные методы
4. Алгебраическая проблема собственных значение
5. Приближение табличных функций
6. Численное интегрирование
7. Решение ОДУ
